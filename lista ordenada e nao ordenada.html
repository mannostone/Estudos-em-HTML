"ul" é a lista não ordenada
"ol" é a lista ordenada

Exemplo não ordenado

  • Arroz
  • Feijão
  • Carne moída
  • Refri

Exemplo ordenado

As listas ordenadas podem ser definidas usando types ("1", "a", "A", "i") em sua tag de abertura

  1. Comer
  2. Comer bastante
  3. Comer a sobremesa
  4. descansar

Lista de descrição -> dl

"dl" representa os limites da lista de descrição
"dt" representa os itens a serem descritos
"dd" representa a descrição de cada item

função Negrito "Strong" utilizada ao destacar as "dt"
